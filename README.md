# code-adventures

code-adventures is a place for me to experiment with different projects and learning activities. I am
the type that likes to be learning about new concepts all of the time, but the one thing I have found
is that they tend to overlap. For example, as I am starting to learn about rendering, I may also look
into ray tracing. The projects do not necessarily overlap, but alot of the math code will. With this
in mind, the code-adventures repository becomes a big monolith where I can develop a toolkit and reuse
the components in future projects without having to manage multiple repositories.
